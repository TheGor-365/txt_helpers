Minimum and Maximum Height and Width:

min-width - это свойство обеспечивает минимальную ширину поля элемента
max-width - это свойство обеспечивает максимальную ширину поля элемента

p {
  min-width: 300px;
  max-width: 600px;
}

---

min-height - это свойство обеспечивает минимальную высоту для коробки элемента
max-height - это свойство обеспечивает максимальную высоту коробки элемента

p {
  min-height: 150px;
  max-height: 300px;
}

Overflow:

hidden - содержимое, которое переполняется, будет скрыто от просмотра
scroll - в поле элемента будет добавлена полоса прокрутки
visible - содержимое переполнения будет отображаться за пределами содержащего элемента

p {
  overflow: scroll;
}

---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Clear:

left  - левая сторона элемента не будет касаться какого-либо другого элемента внутри того же самого содержащего элемента
right - правая сторона элемента не будет касаться какого-либо другого элемента внутри того же самого содержащего элемента
both  - ни одна сторона элемента не будет касаться любого другого элемента внутри того же самого содержащего элемента
none  - элемент может касаться любой стороны

---

div {
  width: 200px;
  float: left;
}

div.special {
  clear: left;
}

---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

position     - позволяет задать положение элемента тремя различными способами
relative     - положение элемента относительно его позиции по умолчанию на странице
absolute     - положение элемента находится относительно его ближайшего расположенного родительского элемента
fixed        - положение элемента может быть закреплено на любой части веб-страницы
z-index      - перекрывает другие элементы
display      - позволяет управлять тем, как элемент перемещается по вертикали и горизонтали документа
inline       - занимают как можно меньше места, и они не могут иметь ручную регулировку ширины или высоты
block        - элементы занимают ширину своего контейнера и могут иметь регулируемую вручную высоту
inline-block - элементы могут иметь заданную ширину и высоту, но они также могут появляться рядом друг с другом и не занимать всю ширину контейнера
float        - позволяет перемещать элементы как можно дальше влево или вправо
clear        -  очистить левую или правую сторону элемента

Grid Auto Flow:

grid-auto-flow - указывает, следует ли добавлять новые элементы в строки или столбцы
row            - указывает, что новые элементы должны заполнять строки слева направо и создавать новые строки, когда элементов слишком много (по умолчанию)
column         - указывает, что новые элементы должны заполнять столбцы сверху вниз и создавать новые столбцы, когда элементов слишком много
dense          - вызывает алгоритм, который пытается заполнить отверстия ранее в макете сетки, если добавляются меньшие элементы

drop-shadow(h-shadow v-shadow blur spread color)	          Применяет эффект тени к изображению.

Возможные значения:
h-тень -требуется. Задает значение пикселя для горизонтальной тени. Отрицательные значения поместите тень слева от изображения.

v-тень -требуется. Задает значение пиксела для вертикальной тени. Отрицательные значения поместите тень над изображением.

размыт ие — необязательно. Это третье значение и должно быть в пикселях. Добавляет эффект размытия в тень. Большее значение
создаст больше размытия (тень становится больше и светлее). Отрицательные значения не допускаются. Если значение не указано,
используется 0 (кромка тени резко).

спред -опционально. Это четвертое значение и должно быть в пикселях. Положительные значения приводят к тому, что тень расширяется
и увеличивается, а отрицательные значения приводят к уменьшению размера тени. Если значение не указано, оно будет равно 0
(тень будет иметь тот же размер, что и элемент).
Примечание: Chrome, Safari и Opera, и, возможно, другие браузеры, не поддерживают эту 4-ю длину; Он не будет отображаться
при добавлении.

Цвет — необязательный. Добавляет цвет к тени. Если не указано, цвет зависит от браузера (часто черного).

Пример создания Красной тени, 8px большой как по горизонтали, так и по вертикали, с эффектом размытия 10px:

фильтр: Drop-Тень (8px 8px 10px красный);
